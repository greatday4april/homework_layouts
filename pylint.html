Pylint and best practices
Pylint is a tool to enforce coding styles and best practices, as well as make sure no basic syntax errors for your python programs. Such as no unused variables, or the code is properly indented etc.
pylint uses .pylintrc to control the settings. To disable some of the warnings from pylint, you can simply add the warning tags to the disable section and those warnings will be suppressed. Be careful not overdoing it though, because that would defeat the whole purpose of using pylint to enforce your coding style.
